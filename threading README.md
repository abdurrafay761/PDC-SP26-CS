Threading

What we learned
Threading allows multiple tasks to run simultaneously within a program.

Code Understanding
This code creates two threads:
- Thread One
- Thread Two

Both threads run at the same time.
The join() method waits for threads to finish.

How to Execute
python threading_example.py

End Use
- Download manager
- Background processing
- Web servers

When to Use
When multiple tasks need to run at same time

How to Use
Use threading module  
Create thread  
Start thread  
Join thread  

Advantages
- Fast execution
- Multitasking
- Efficient CPU usage

Disadvantages
- Race condition
- Debugging difficult
- Deadlock risk
